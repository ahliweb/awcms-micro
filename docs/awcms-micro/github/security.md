## Catatan Bun

Dependabot dikonfigurasi dengan `package-ecosystem: "bun"` karena repo menggunakan `bun.lock` text lockfile dan `packageManager: "bun@1.3.14"`. Jangan mengganti ke `npm`, `pnpm`, atau `yarn`.
